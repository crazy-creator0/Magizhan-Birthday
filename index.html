🏍️ Open Rapido to Venue
⚡ Zero-tap mode — Tasker + AutoInput auto-types the destination & selects it for you. One-time setup required.
1
Install Tasker and AutoInput from Play Store. Both need to be licensed (paid or trial).
2
Enable AutoInput Accessibility Service: Settings → Accessibility → Installed apps → AutoInput → Turn on.
3
In Tasker: tap ☰ → Backup, then import the XML file shared with you (task: "Rapido – Go to Venue").
4
Open Rapido once manually and check the exact text on the destination box — it may say "Where to?" or "Drop location". Update Step 3 of the task (ActionId field) to match that exact text.
5
Done! Now long-press the Rapido button in this page (hold 1.2 sec until it says "Tasker Activated ⚙️"), then release — it will fully automate the booking.
💡 If Tasker step fails: The most common reason is the destination search box text doesn't match. Open Rapido, read the exact placeholder text, and update ActionId in Step 3 of the task to match it.
1
Tap "Open Rapido" below — the app will launch.
2
In Rapido, tap the destination search box (says "Where to?" or "Drop location").
3
Tap 📍 Select on map (shown below the search bar).
4
Tap the search field on the map screen and paste the coordinates — the pin will drop at the exact venue!
